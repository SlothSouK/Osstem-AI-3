# /sync — CLAUDE.md 갱신 + GitHub 푸시

이 커맨드를 실행하면 다음 두 작업을 순서대로 수행합니다:

## 1단계: CLAUDE.md 갱신

현재 프로젝트 상태를 파악하고 CLAUDE.md를 최신 내용으로 업데이트합니다.
확인할 항목:
- 새로 추가된 파일/폴더가 있으면 디렉토리 구조에 반영
- 새로운 실행 방법이나 설정이 생겼으면 해당 섹션 업데이트
- 기술 스택이나 의존성이 바뀐 경우 반영
- 삭제/이동된 내용은 제거

## 2단계: 변경사항 커밋 & GitHub 푸시

1. `git status`로 변경된 파일 확인
2. 변경된 파일 전체를 스테이징 (`git add`)
   - 단, `.env`, `lottery.html` 등 .gitignore 파일은 제외
3. 변경 내용을 요약한 커밋 메시지 작성 후 커밋
4. `git push origin main` 으로 푸시

완료 후 어떤 파일이 커밋되었는지 요약해서 알려줍니다.
